## About

I'm an undergraduate student interested in the intersection of:

- **Generative Models**
- **Mathematical structure in ML**
- **Perceptual color + visual fidelity**
- Systems that behave nicely *in theory* and *in practice* (rare)

Right now, my main focus is on improving **color consistency and control in Latent Diffusion Models**, especially using perceptual losses in RGB/Lab spaces.

I enjoy building things, breaking them, and then writing loss functions about it.
